Zender 3D was designed to reach scientific visualization and 3D editing using the Visualization Toolkit in Python, similar to Blender. The idea was to make an interactive 3D editor capable of importing, viewing, updating, transforming, and saving scientific and geometry 3D models. In principle, it can do what other 3D modeling tools can do. Zender 3D utilizes VTK and PyQt5 for its user interface, handling of user interaction, and execution of the system. Full featured Zender 3D can deal with simple and complex 3D jobs. The following are the import options such as basic forms, cell-based models, VTK models, implicit surfaces, 3D functional data, and multi-actor 3D Studio scenarios. The user is allowed to translate, rotate, scale, copy, and delete a model, as well as manage model visibility. An object centered transformation box widget and trackball camera allow you to select, interactively pick, and change objects in the 3D model editor. Zender 3D is a powerful, command design principle-based undo/redo system, which keeps a history of the user's actions and offers them a lot of freedom in editing models during one session.

The UI was simple, intuitive, and easy to use. It uses PyQt5-considered a modern, straightforward layout. First, there is a splash screen follow with welcome message, in the system, it displays a side scene navigator with buttons and menus for loading models, changing visual settings, and controlling your workspace without confusion. Messages both in the terminal and GUI let a user know what's going on such as a model is loaded, an item is selected, a transformation has occurred, or a camera view has changed. A grid and orientation axes in the renderer describe object positions, making it easier to understand the movement of 3D objects. Saving and opening scenes is another feature of Zender 3D. Each model's position, rotation, and colour are stored in a folder with VTK PolyData and a scene JSON. The user does not have to start over. The importing of your saved scene into the editor brings everything back, camera view and all. Shortly, Zender 3D will be more fluid like other real 3D editing software that allows the user to create a project and edit it over and over. In conclusion, Zender 3D fulfills the main goals for this project task, it is almost a complete 3D editing software and shows how VTK would render dynamic, adaptive, and easy scientific visualization tools. Zender 3D, just like Blender, simplifies the operations of loading, transformation, and rendering.
