The goal of the first assignment is to accustom you with OpenGL and PyQt5 capabilities. You must apply the object-oriented programming paradigm to create a 2D graphics system called "MCG: The Clock Wall". Your created graphics application must utilize PyOpenGL and its associated libraries.
