The mini-project for this semester requires the development of an engaging and interactive Bouncing Ball Game. Once again, a basic bouncing ball source code in C will be provided, and it needs to be converted into a Python environment. That is not enough, the source code must be improvised with a number of gameplay features, scoreboarding, 3 levels of difficulties, and a nice interface landmark to attract users playing the game. Bonus points will be awarded to groups that are able to produce highly engaging games with an excellent game plan and level of difficulties. This mini-project requires the team to develop a three-level game with sufficient interactivity. The team may come up with diverse and engaging principal interfaces that match the game's concept and gameplay.
