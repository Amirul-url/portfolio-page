This project focuses on the development of an integrated image enhancement application that addresses the task assigned in enhancing hazy images through advanced image processing techniques. The four major tasks involved in the job requirements are image preprocessing, including noise removal, sharpening, and histogram equalization, implementation of a Dark Channel Prior (DCP) method for haze removal to show the details in an image, enhancing images with High Dynamic Range (HDR) processing to balance brightness and contrast, and combining or modifying DCP and HDR techniques for superior results. It contains a GUI that allows users to load any image, save it after processing, crop a portion of it, and view the real-time histogram of both the original and enhanced images. The system also has slider parameters that adjust noise levels and the strength of enhancement. This system is implemented using Python with an OOP approach. The following libraries are used OpenCV, NumPy, and Matplotlib. The performance of the enhancement will be measured in terms of Structural Similarity Index (SSIM) and Peak Signal-to-Noise Ratio (PSNR) to ensure that the enhancement has a measurable improvement in terms of similarity and quality of the processed images. This mini project will effectively and efficiently show how to restore the quality of hazy images in a user-friendly way by integrating these techniques into a cohesive application, thus addressing the requirements of the assigned task.
