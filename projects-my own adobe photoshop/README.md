Your goal for this assignment is to further enhance your previous 2D Image Editor. For this assignment, I only consider the following features to be evaluated. No first assignment 2D Image Editor will be counted. New features you should add to your own Adobe Photoshop are:

-Able to create a thumbnail image for every loaded or modified images as a special button for your editor.

-Able to create three or two drawing windows to assist users manipulating their loaded images side by side together with gridlines or ruler features.

-Able to display a real-time histogram panel of the loaded original images and modify images. The real – time histogram panel has the capability to differentiate 3 distinct channels (RGB) and combine channel. Provide additional button to alternate between these two features.

-Able to enhance the quality of image by applying 6 filtering methods. Don’t forget to use sliders panel/components to adjust every filter property.

-Able to perform Bit-plane slicing technique and display the output as an array of bit plane images using plt.show () and cv2.imshow(). Provide the features of the bit-plane slicing features.

-Able to detect edges by using Canny Edge Detection, Prewitt Edge Detection and Sobel edge detection techniques. Don’t forget to use sliders panel/components or any suitable panel/button to adjust every property of each Edge Detection Technique.

-Able to perform image contouring process.

-Able to apply image thresholding interactively by using slider.

-Able to apply Power Law Transformation interactively by using slider.

-Able to perform Piecewise Linear Transformation interactively by using slider.

-Able to perform image erosion and dilation process.

-Able to apply histogram equalization method.
