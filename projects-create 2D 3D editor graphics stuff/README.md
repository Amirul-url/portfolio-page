Your second assignment is to further improve your understanding about the strength and capabilities of OpenGL libraries in producing a realistic image synthesis. You will be provided with several basic OpenGL source codes in C; you must convert them all to Python and create the drawing work using any of the PyOpenGL compatible commands. That is not enough, you must improvise the output so that it is amazing and beyond its standard output.
